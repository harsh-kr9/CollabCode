# CollabCode

A real-time collaborative code editor platform with integrated video chat, team messaging, and an AI coding assistant.

## Features

- Real-time collaborative code editing with Monaco Editor
- Video and voice calls using WebRTC
- Team chat with message history
- AI coding assistant powered by Google Gemini
- Multi-language code compilation (Python, C++, Java, JavaScript)
- Project management dashboard
- Google authentication via Firebase

## Tech Stack

- **Frontend:** React.js, Tailwind CSS, Redux Toolkit, Socket.IO
- **Backend:** Node.js, Express.js, Socket.IO
- **Database:** MongoDB
- **AI:** Google Gemini via LangChain
- **Auth:** Firebase Authentication
- **Deployment:** Vercel (Frontend), Render (Backend)
